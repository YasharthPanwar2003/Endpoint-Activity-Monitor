******Python Endpoint Activity Monitor******
****Project Overview****

A modular desktop application designed to track and log user input events. Built using Object-Oriented Programming (OOP) to demonstrate system monitoring, data persistence, and UI separation in Python.

#For educational and cybersecurity research purposes only

****Functional Features****

✓ **Real-time Monitoring:** Captures keyboard inputs in the background.

✓ **Data Buffering**: Stores logs in memory to optimize performance before saving.

✓ **Multiple Exports:** Saves data to TXT, CSV, or JSON formats.

✓ **Session Control:** Start, Stop, and Clear logs via a GUI.

✓ **Clean Architecture**: Separates logic (Backend) from display (UI).

****Technologies Used****

**Language:** Python 3

**Library:** pynput (Input monitoring)

**Interface:** Tkinter (GUI)


****How to Run****

1. Install the dependency:
   
   **pip install pynput**
   
2. Run the application:
   
   **python main.py**
